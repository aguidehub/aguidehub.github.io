Step 1: Add a quick selection tool.

To begin, let‚Äôs choose the Quick Selection Tool. This can be done by clicking on the icon depicted below located in the toolbar down the left side of the screen, or by hitting the B button on your keyboard.If you want to select image area.

Step 2: Rifine-edge.

Go to the options menu bar and click on Refine edge.

Click on the Refine Radius Tool and select the outer image area. Go to Refine edge -> adjust-edge and enter the following values:

  • Smooth: 13
  • Feather: 1.9
  • Contrast: 13
  • Shift edge: 0

Go to the add layer mask at the bottom of the tools panel.

Step 3: Apply to add a layer mask.

You can see open the add layer mask layer available on Photoshop right side section.

Then select add layer mask and click on delete icon and pop-up delete box and click on apply.

Step 4: New page.

Go to File -> click on -> New and set width 1200, height 1200.

Can you see the white page and drag and drop your image here.

Step 5: Apply the cutout edges effect.

Go to Filter click on filter-gallery.

Click Filter -‚Ä∫ Filter Gallery -‚Ä∫ Artistic: Cutout Edges. From here, you can set a high Edge Thickness, set the Edge Intensity to a low value or add a low Posterisation value depending on the image.
