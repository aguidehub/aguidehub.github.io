How to add chromatic aberration in photoshop?

April 09, 2022

Hello Friends 👋,

Welcome To aGuideHub! ❤️

This may be the best Photoshop chromatic aberration an article, so if you want to learn the step-by-step process, keep reading!

Table of contents

  • Insert image.
  • Select duplicate the layer.
  • Blending options.
  • Close the background layer eye.
  • Click on the thumbnail.
  • Select the move tool.

Let’s start with the first step.

Step 1: Insert image.

This will open a file browser or download folder you can select the image. And click the open button. Go to File -> Open or press CTRL + O

How to add images in photoshop?

Press Ctrl + 5 to turn the image black and white

Step 2: Select duplicate the layer.

To duplicate the layer, select the layer and press ctrl+j.

Step 3: Blending options.

You can go to the bottom right of the screen, click on the small FX button, and select blending-options from the pop-up menu.

A window will pop up and select red(R) and deselect green(G) and blue then click on ok button.

step 4: Close the background layer eye.

Close the background layer eye, you can see the image is red color.

Select layer 1 and press ctrl+j to dupicate.

You can go to the bottom right of the screen, click on the small FX button, and select blending-options from the pop-up menu. A window will pop up and deselect red(R) and select green(G) and blue(B) then click on ok button.

Close the background layer eye, layer 1 eye, you can see the image is green color.

Step 5: Click on the thumbnail.

Select layer 1 copy hold ctrl and click on the thumbnail.

Step 6: Select the move tool.

The move tool is selected. Tab the arrow keys to move the layer to the opposite side of layer 1.

Tab the left or right arrow key to move the layer you can see the effect appear.

Finally, we have created a chromatic aberration in image.
