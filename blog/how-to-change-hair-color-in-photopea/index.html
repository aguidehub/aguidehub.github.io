To change hair color in photopea, select on the lasso tool and click on new adjestment layer -> hue-saturation select the hair area in which you want to change hair color. It will change the hair color.

Today, I am going to show you how I change hair color in photopea.

Table of contents

  • New page.
  • Insert image.
  • Select a lasso tool

Let‚Äôs start with the first step.

Step 1: New page.

Go to Photopea.com to start a free online photo editor. And create a new page.

Step 2: Insert image.

Open a photo from your computer: click File - Open (in the top left corner), then find and select the file in your device (or use drag and drop).

Step 3: Select a lasso tool.

Select a lasso Tool from the toolbar on the left side.

Click into the image and drag the mouse

Click on new adjestment layer -> hue-saturation.

Set the hue saturation and change the color.

Finally, here we have changed the hair color.

When you follow the above steps then it will change hair color and it will show on the image in photopea.
