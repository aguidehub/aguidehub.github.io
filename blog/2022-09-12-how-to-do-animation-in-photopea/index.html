How to do animation in photopea?

September 12, 2022

Hello Friends 👋,

Welcome To aGuidehub! ❤️

To do animation in photopea, go to layer ->animation -> select the make frames ,and then go to file -> export as -> click on gif, It will make animation.

Today, I am going to show you how I do animation in photopea.

Table of contents

  • Insert image.
  • Create a duplicate-layer.
  • Create a animation.
  • Click on export-as-gif

Let’s start with the first step.

Step 1: Insert image.

Open the photopea and Select photo, it will open a file browser or download folder you can select the image. And click open button.

Go to the layer panel, press the right mouse on the layer, open the popup box select duplicate layer.

Step 2: Create a duplicate-layer.

Click on a layer and create many layers of one image.

Press on the shift button and then select all layer.

Photopea will find all folders, and merge their frames together. Alternatively, you can select only folders you want to make frames (in Layers Panel) before pressing Layer - Animation - make frames.

Step 3: Create a animation.

Photopea detects all layers starting with a. The first such layer is shown in the first frame (others starting with a are hidden). The second such layer is shown in the second frame, etc.

Go to file -> export as -> click on gif, It will make animation.

Step 4: Click on export-as-gif.

After clicking on the gif, the box for save for web will open. then you here can see animation

When you follow the above steps then it will do animation in photopea.
