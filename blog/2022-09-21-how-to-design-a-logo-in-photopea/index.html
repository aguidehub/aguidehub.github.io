How to design a logo in photopea?

September 21, 2022

Hello Friends 👋,

Welcome To aGuidehub! ❤️

To design a logo in photopea, everyone has different creativity to the creative logo so here, we will learn the most straightforward way to design a unique logo based on your brand name. let’s get started.

Today, I am going to show you how I design a logo photopea.

Table of contents

  • Choose the rectangle-select.

Let’s start with the first step.

Step 1: Choose a type tool.

Select the type-tool, This can be done by clicking on the icon depicted below located in the toolbar down the left side of the screen, or by hitting the t button on your keyboard.

Create a logo.

Step 2: Click on merge down.

Select the G layer and click on the right mouse button then it will open a popup, select merge down, it makes two-layer of the one layer.

Step 3: Choose the rectangle-select.

Go to the toolbar and select the rectangle-select.

Step 4: Select the move tool.

Select the rectangle and click on the image and then set the text in the middle with the move tool.

Step 5: Go to layer -> deselect.

Go to the menu bar and click on the layer then select the * to deselect*.

Step 6: Click on create-new-layer icon.

Select the layer on top of which you want to create a new layer and click the Create New Layer icon.

Step 7: Choose the brush tool.

Go to the toolbar and select the brush-tool.

Set the brush size on 586 px.

Click on the page.

Step 8: Insert pattern.

Select a new layer and put it on your page

Step 9: Click on free-transform.

Go to the menu bar and click on the edit then select the free-transform.

Step 10: Select the clipping mask.

First, select the layer and click on the mouse button and it will open a popup then select clipping mask.

Step 11: Set the pattern.

Then set the pattern in the text.

Step 12: Return click on the create-new-layer icon.

Select the layer on top of which you want to create a new layer and click the Create New Layer icon.

Step 13: Go to layers palette -> new adjesment layer-> fill.

Go to the layers palette and click the mouse button on new adjustment layer and it will open the popup and then create a fill...

Select the layer 2 and click fill then change the color.

Step 14: Select the clipping mask.

First, select the layer and click on the mouse button and it will open a popup then select clipping mask.

Select the layer on top of which you want to create a new layer

When you press the ctrl keypad and click the mouse on the layer, the text will be selected.

Step 15: Go to edit -fill.

Select layer 3 and go to menubar click on edit -fill then it will open the dialog box.

Then open a dialog box set the black color and press the ok button.

Step 16: Go to edit -fill.

Here close the eye of the main layer, then you will see text in black color.

Step 17: Create a duplicate layer.

Select layer 3 and create a duplicate layer.

Then click the mouse on the text and shake it here and there. After that put the top 4 layers in 1 folder.

Step 18: Set the blending layer.

Select layer 1, choose blending mode, and click on stroke, set the opacity is 100 px.

Select layer 1, choose blending mode, and click on stroke, set the size is 27 px.

When your project is completed, go to the file and click export as and save it in png.

Step 19: Save the export-as.

Here we have created design a logo in png from.

When you follow the above steps then it will design a logo in photopea.
