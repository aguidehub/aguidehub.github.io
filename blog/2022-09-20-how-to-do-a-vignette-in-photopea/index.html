how to do a vignette in photopea?

September 20, 2022

Hello Friends 👋,

Welcome To aGuidehub! ❤️

To do a vignette in photopea, go to the right panel, click on blending options, Select the gradient overlay and It will vignette the image.

Today, I am going to show you how I do vignette photopea.

Table of contents

  • Insert images.
  • Click on blending options.
  • Set the layer style
  • Return Click on blending options.
  • Set the blend mode.

Let’s start with the first step.

Step 1: Insert images.

Open the photopea and Select photo, it will open a file browser or download folder you can select the image. And click the open button.

Step 2: Click on blending-options.

Going to the right side of the photopea, in the layer panel, first select the layer, and click on blending options.

Step 3: Set the layer style.

When you click on the blending option the layer style will open, first click on gradient overlay, and then check mark reverse, and check mark align with layer, angle set 60%, scale set 175%.

Step 4: Return click on blending-options.

Going to the right side of the photopea, in the layer panel, first select the layer, and click on blending options.

Step 5: Set the blend mode.

When you click on the blending option the layer style will open, and set the blend mode-> multiple.

Everything will be the same, and set the blend mode-> multiple, and click on the ok button.

Here we have created a vignette on the image.

When you follow the above steps then it will vignette image in photopea.
