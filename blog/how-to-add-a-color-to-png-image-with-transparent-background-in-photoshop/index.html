Once you have a PNG image with a transparent background, adding a background color is a quick process in Photoshop.

Table of contents

  • Insert image.
  • Add solid fill layer.
  • Select your background color.
  • Move layer below.

Let‚Äôs start with the first step.

Step 1: Insert image.

This will open a file browser or download folder you can select the image. And click open button. Go to File -> Open or press CTRL + O

To add a image guide an article: How to add images in photoshop?

The checkered background shown below is Photoshop‚Äôs way of telling you that the file has a transparent background i.e. no background.

Step 2: Add solid fill layer.

Add a Fill Layer by going to Layer > New Fill Layer > Solid Color.

If your image layer has a lock symbol to the right, click on it to unlock it. This will let you move the new layer to be below the image layer. This used to be the case in older versions of Photoshop.

In the dialog box that appears, click ok.

Step 3: Select your background color.

In the color dialog box that opens up next, use click and drag the white sliders to select the general color palette, then click and drag to move the circle to the specific color you want. Click ok.

Step 4: Move layer below.

When you add a fill layer, by default the fill layer appears above the image layer.

You need to click and drag the fill layer and move it below the image layer.

If you want to change the background color, you have to double-click on the layer thumbnail. Then one dialog box will open and choose the color of your choice.

Lastly, we changed the color of the transparent background.

when you follow the above steps then it will create color to png images with a transparent background and will show on the new background color available on photoshop.
