Step 1: Add A black color fill layer.

Open image in photoshop,we will start by filling our image with black, which will become the color of the vignette border.click on the small Create Fill or Adjustment layer icon, and select Solid color from the pop-up menu.

You choose Solid Color from the list, Photoshop pops open the Color Picker so we can select the color we want to use. Choose black color, By entering 000 for the R, G, and B values ‚Äã‚Äãin the large color selection box on the left or near the bottom center of the dialog box:

Click OK to close out of the Color Picker.

Step 2: Select the layer mask.

What Photoshop named ‚ÄúColor Fill 1‚Äù is sitting on top of the original image on the Background layer, and we see the layer mask thumbnail directly to the left of the layer name.We will select the layer mask.

Step 3: Select the elliptical marquee tool.

Select the Elliptical Marquee Tool from the Tools panel. So click on the Rectangular Marquee Tool and hold your mouse button down for a second or two until a fly-out menu appears showing the other tools available in that spot, then choose the Elliptical Marquee Tool from the list:

Step 4: Elliptical selection outline.

With the Elliptical Marquee Tool selected, click inside the image and drag out an elliptical selection outline in the center of the image.

Step 5: Select with black fill.

Go up to the Edit menu in the Menu Bar along the top of the screen and choose Fill. Go to Edit -> Fill..

This opens the Fill dialog box. Change the Use option at the top of the dialog box to Black.

Click OK to close out of the Fill dialog box. And see the image on the selected area. The layer mask thumbnail in the Layers panel shows us the area we filled with black.

Deselect The Selection

Go up to the Select menu at the top of the screen and choose Deselect. Go to Select -> Deselect.

Step 6: The gaussian blur filter.

To soften the edges of the vignette border, we‚Äôll use Photoshop‚Äôs Gaussian Blur filter. Go up to the Filter menu at the top of the screen, choose Blur, then choose Gaussian Blur. Go to Filter -> Blur -> Gaussian Blur.

This opens the Gaussian Blur dialog box. And see the Radius slider at the bottom of the dialog box towards the right. Than set 37.4px or as you like, click on ok.

Step 7: Reshape with free transform.

Go up to the Edit menu at the top of the screen and choose Free Transform. You can also press Ctrl+T to quickly access Free Transform with the keyboard shortcut. You can resize, the free transform tool. Go to Edit -> Free Transform.

you will select Free Transform, you will see like this below

Step 8: Trim away any extra black space.

If you want to trim any extra black space around the image, go to the Image menu at the top of the screen and select Trim. Go to Image -> Trim.

This opens the Trim dialog box. Select the Top Left Pixel Color option in the top half of the dialog box if it‚Äôs not already selected, and leave the Top, Bottom, Left, and Right options selected in the bottom half of the dialog box.

Click OK to close the dialog box, and Photoshop removes any extra black spaces around the image, giving us our final Vignette result.
