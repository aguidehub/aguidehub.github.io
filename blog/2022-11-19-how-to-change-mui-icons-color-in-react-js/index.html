How to change mui icons color in react js?

November 19, 2022

Hi Friends 👋,

Welcome To aGuideHub! ❤️

To change mui icons color, set in style attribute sx={{ color: "#0089ff" }}. It will change mui icons color.

Today, I am going to show you, how to change mui icons color in react js.

Installation

Install the following packages to use mui icon in react js.

npm

npm install @mui/material @emotion/react @emotion/styled

yarn

yarn add @mui/material @emotion/react @emotion/styled

MUI material change mui icons color example.

App.js

import * as React from 'react';
import Stack from '@mui/material/Stack';
import IconButton from '@mui/material/IconButton';
import ArrowForwardIcon from '@mui/icons-material/ArrowForward';

export default function IconButtonSizes() {
  return (
    <Stack direction="row" alignItems="center" spacing={1}>
      <IconButton aria-label="delete" sx={{ color: "#0089ff" }}>
        <ArrowForwardIcon />
      </IconButton>
    </Stack>
  );
}

In the above code example, I have used the @mui/material component and changeed mui icons color of MUI.
