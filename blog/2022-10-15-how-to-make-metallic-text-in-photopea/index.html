How to make metallic text in photopea?

October 15, 2022

Hello Friends 👋,

Welcome To aGuidehub! ❤️

To create metallic text in photopea first select type tool and type text and choose the color #ffab27. Then choose bevel-and-emboss, it will make metallic text in photopea.

Today, I am going to show you how I make metallic text in photopea.

Table of contents

  • Open a new project.
  • Choose the type tool.

Let’s start with the first Method.

Step 1: Open a new project.

To change the background color, click on a file from the menu bar and then select new or use the keyboard shortcut alt+ctrl+n a new project window will appear.

Double-click on the layer, and it will open a dialog box, click on gradient overlay, then click on the gradient and it will open gradient editor, click on the color panel. Now a color picker window will appear you can change the color #949494 from here just select a color and click ok

When you double-click on the layer, it will open the layer style then set the gradient, checkmark the reverse, set the angle to 90%, scale 178%, then click the ok button.

Step 2: Choose the type tool.

Select the type-tool from the toolbar on the left side of your workspace. Wright a text.

When you write a text then you will change the font style and font size.

When you select the type tool, you can see the color box in the top menu bar, then click it, it will open the color picker window then # ffab27 select the color ok button.

Then double-click on the text layer, it will open a window, click on the bevel and emboss, and set the styles -> inner bevel. Technique -> smooth up Depth -> 500% Size -> 30px Soften -> 2 px Angle -> 120% 30%. Click on the ok button.

This will open a window and click on Contour, setting the range to 50%.

Final result.

When you follow the above steps then it will make metallic text in photopea.
